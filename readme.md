# задание 1
## задача 2
### текст задачи
Получить список из последних цифр чисел, содержащихся в исходном списке
### Алгоритм решения
1) спрашиваем у пользователя количество элементов в списке
2) при помощи цикла просим пользователя ввести данное кол-во элементов
и добавляем их в список
3) при помощи list.map применяем к каждому элементу списка функцию которая находит
последнюю цифру
4) выводим полученный список

# задание 2
## задача 2
### текст задачи
Найти сумму тех элементов списка, которые оканчиваются на заданную цифру
### алгоритм решения
1) спрашиваем у пользователя количество элементов в списке
2) при помощи цикла просим пользователя ввести данное кол-во элементов
и добавляем их в список
3) пользователь вводит цифру для поиска
4) при помощи List.fold проходим по списку и накапливаем сумму только тех
элементов, последняя цифра которых совпадает с введенной пользователем
5)выводим данную сумму
